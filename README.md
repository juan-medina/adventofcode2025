# Advent of Code 2025

This repository contains solutions for [Advent of Code 2025](https://adventofcode.com/2025), implemented in TypeScript.

## Project Structure

- `src/` - Source code for each day's challenge
  - `days/` - Individual day solution files (e.g., `day01.ts`)
  - `index.ts` - Entry point for running solutions
- `data/` - Input files for each day (e.g., `01.txt`)
- `test/` - Unit tests for each day's solution

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or newer recommended)
- [npm](https://www.npmjs.com/)

### Install Dependencies

```
npm install
```

### Running a Solution

To run the solution for a specific day (e.g., Day 1):

```
npm start -- 1
```

### Running Tests

```
npm test
```
